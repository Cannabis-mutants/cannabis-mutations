# Cannabis Mutations Dataset — Data Dictionary

**Version 0.1 · Licence: CC-BY 4.0 · Source: Cannabis Mutants (https://cannabismutants.com)**

A machine-readable dataset of documented *Cannabis* leaf-morphology mutations.
Files: `cannabis-mutations.csv`, `cannabis-mutations.json` (identical records).

## Fields

| Field | Description |
|---|---|
| `slug` | URL-safe identifier for the mutation. |
| `name` | Common name of the mutation. |
| `section` | Site section / top-level category it belongs to. |
| `morphology_group` | Broad taxonomic group (Leaflet & blade, Phyllotaxy & architecture, Pigmentation, Chromosomal, Seed & seedling). |
| `also_known_as` | Synonyms and alternate names in circulation (semicolon-separated). |
| `definition` | One-line definition of the morphology. |
| `genetic_basis` | Genetic or developmental cause, where known. |
| `inheritance_stability` | Whether/when the trait is stable. |
| `expression_rate` | How often it appears (e.g. Mendelian ratios). |
| `origin` | Reported origin/discovery. |
| `first_documented` | Approximate first documentation. |
| `reproducibility` | Whether it breeds true / can be propagated. |
| `notable_traits` | Distinguishing features and effects. |
| `references` | Sources consulted (secondary; see note). |
| `url` | Canonical page for the mutation. |
| `license` | Licence for the record (CC-BY-4.0). |

## Notes on rigour
This dataset reflects the current documented record for a young, hobbyist-driven field.
Much source material is grower literature rather than peer-reviewed science; the `references`
are starting points, not primary genetics papers. Fields flagged as "not fully understood"
or "not yet characterised" mark genuine gaps. Corrections and primary sources are welcome.
