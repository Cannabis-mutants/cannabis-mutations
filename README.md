# Cannabis Mutants — Open Data & Taxonomy

Open, freely-reusable reference data for **cannabis mutations** — a young, rising and
under-documented field. Maintained by **Cannabis Mutants** (https://cannabismutants.com).

Everything here is released under **CC-BY 4.0**: reuse it, including commercially, with
attribution to Cannabis Mutants (cannabismutants.com).

## Contents
- **`leaf-morphology-taxonomy.md`** — the classification and plain-language definitions of
  each leaf morphology / mutation type.
- **`cannabis-mutations.csv`** / **`cannabis-mutations.json`** — machine-readable dataset of
  documented mutations.
- **`DATA-DICTIONARY.md`** — field definitions for the dataset.

## Why open?
Cannabis mutations live mostly as scattered grower folklore under inconsistent names.
Open, shared data + a common vocabulary lets everyone build on the same foundation instead
of re-inventing it — and lets researchers, developers and other sites reuse and cite it.

## How to cite
> Cannabis Mutants (2026). *Cannabis Leaf Morphology & Mutation Taxonomy* / *Cannabis Mutations Dataset*, v0.1. cannabismutants.com. CC-BY 4.0.

## Contribute
Corrections, better/primary sources, photographs, and newly documented morphologies are
welcome. This is a living reference for a field that is only just getting a standard.
